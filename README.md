## 快速上手

- 推荐预先安装 `nodejs 16`

```bash
# clone the project
git clone https://github.com/786744873/easy-dotnet

# enter the project directory
cd easy-dotnet

# install dependency
npm install --registry=https://repo.huaweicloud.com/repository/npm/ # or yarn install

# develop
npm run dev # or yarn dev
```
